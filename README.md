# BrokeChef Android Mobile App

## About The Project

Android Mobile App written in Kotlin for this Website: [BrokeChef](https://github.com/Tomas178/BrokeChef)

## Features
- Sign up/Sign in
- Add/update profile image
- Create recipe (User can add his own image or an AI will generate based of ingredients and title)
- Delete recipe
- Save/unsave recipe
- Rate/remove rating
- Mark/unmark recipe as cooked/tried
- Create/remove collections
- Add/remove recipes to/from collections
- FRIDGE MODE: Add your fridge image and get 3 fully generated recipes. Then select 1 of them and it will be created. (MAXIMUM 5 REQUESTS PER 60 MINUTES FOR USER)
- Follow/unfollow another user
- Look through following/followers list of a user
- Search recipes in homepage
- Search recipes by search bar
- Sorting (newest/oldest, highest/lowest rated and also recommended by users saved recipes)
